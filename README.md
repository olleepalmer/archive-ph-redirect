# Archive.ph Redirect

Chrome extension that automatically opens archived versions of articles via archive.ph.

Useful for accessing cached copies of pages that may have changed, been removed, or are otherwise unavailable.

## Install

1. Download/clone this repo
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the folder

## Usage

- Redirects happen automatically for article pages
- Click extension icon to toggle on/off or edit domain list
